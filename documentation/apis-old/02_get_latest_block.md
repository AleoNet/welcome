
```bash title=ENDPOINT
GET /{network}/latest/block
```

Returns the latest block.

### Arguments

None

### Response
| Parameter |                 Type                  |   Description    |
|:---------:|:-------------------------------------:|:----------------:|
| `result`  | [object](../concepts/fundamentals/05_blocks.md) | The latest block |
