```bash title=ENDPOINT
GET /{network}/stateRoot/latest
```

Returns the latest state root.

### Arguments

None

### Response
| Parameter |                 Type                  |      Description      |
|:---------:|:-------------------------------------:|:---------------------:|
| `result`  | [string](../../concepts/fundamentals/05_blocks.md) | The latest state root |
