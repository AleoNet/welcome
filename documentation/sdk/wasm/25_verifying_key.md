<a name="VerifyingKey"></a>

## Overview

<p>Represents a verifying key for a function within an Aleo program. The VerifyingKey class provides methods for loading, serializing, and inspecting verifying keys which are used to verify zero-knowledge proofs generated by program executions.</p>
